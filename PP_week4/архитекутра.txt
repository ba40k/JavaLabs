Классы в программе есть работающие с данными от пользователя и с данными от других классов. Первые могут кидать исключения.
Главный класс Lab - просто вызывает нужные методы для выполнения лабы.
Класс BinarySearchTree - состоит из Node, парамтризуется T, которое доолжно реализовывать функцию сравнения.
У этого класса есть методы для выполнения каждого из запросов.
Для выполнения обхода используются классы, реалищующие интерфейс TraversalStrategy, в нем только один метод traverse, который выполняет обход. Соответсвтенно разные классы выполняют разные обходы.
Для них предусмотрена фабрика, строящая нужный тип обхода на основе ввода.
Класс Query - это класс запроса, который выполняет дерево. От него наследуются классы конкретных запросов. Для них предусмотрена фабрика, которая строит нужный запрос по строке из ввода. Сам же класс использует паттерн комманда и реализует интерфейт BinarySearchTreeCommand, имеет единсвтенный метод execute(BinarySearchTree tree),где вызывает нужнй метод
